### **Project Overview**

Synca is a full-stack web application designed to simplify the process of finding and booking Paying Guest (PG) and hostel accommodations. It aims to create a centralized, reliable platform that connects property owners with students and young professionals.

The key differentiator for Synca is its granular room and bed-level booking system, which will allow users to view real-time availability and select specific beds. The platform also includes a robust management portal for PG owners to handle both online and offline bookings seamlessly.

### **Key Features**

#### For Students (Tenants)
- [ ] User Registration and Login
- [ ] Advanced Search and Filtering for PGs (by area, price, amenities)
- [ ] Detailed PG Information Pages with Photos and Reviews
- [ ] Visual Room and Bed Selection UI
- [ ] Secure (Simulated) Booking and Payment Flow
- [ ] User Profile with Booking History
- [ ] Ability to Post Ratings and Reviews

#### For PG Owners
- [ ] Owner-specific Registration and Login
- [ ] Create and Manage PG Property Listings
- [ ] Add and Manage Rooms and Beds for each Property
- [ ] Visual Occupancy Dashboard to Track Bookings
- [ ] Handle Online & Offline Bookings (via Django Admin)

### **Technology Stack**

* **Backend:** Python, Django Framework
* **Backend:** Python, Django Framework
* **Database:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Development Environment:** Python Virtual Environment (`venv`), Git
